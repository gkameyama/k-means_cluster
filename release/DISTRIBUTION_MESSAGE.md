# 配布文面サンプル

以下は、そのままメールやチャットに貼れる配布文面の例です。

---

K-Means クラスタリングツールをお送りします。

添付ファイル:

- `KMeansClusterGUI.exe`
- `USER_MANUAL.md`

このツールは Windows 用です。入力ファイルとして `.csv` / `.xlsx` を読み込み、クラスタリング結果を同じ形式で保存できます。

出力先はファイルではなくフォルダを指定します。保存されるファイル名は `（入力ファイル名）_result_月日時分` 形式で自動生成されるため、元データや既存結果を上書きしません。

使い方は `USER_MANUAL.md` をご確認ください。

補足:

- 初回起動時は少し時間がかかることがあります。
- セキュリティソフトの確認メッセージが表示された場合は、内容をご確認のうえ実行してください。
- ご不明点や不具合があればご連絡ください。

---

## 短い版

K-Means クラスタリングツールをお送りします。Windows 用の実行ファイルです。

入力は `.csv` / `.xlsx`、出力も `.csv` / `.xlsx` に対応しています。出力先フォルダを指定すると、`（入力ファイル名）_result_月日時分` の名前で保存されます。使い方は同梱の `USER_MANUAL.md` をご確認ください。

初回起動時は少し時間がかかる場合があります。ご不明点があればご連絡ください。
